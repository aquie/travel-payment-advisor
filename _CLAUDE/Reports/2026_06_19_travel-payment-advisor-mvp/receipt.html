주요 결정

  • 실제 환율처럼 보이는 기본값이나 확인되지 않은 환율 API를 사용하지 않는다.
  • Toss 2%는 현재 공식 확인된 2026-09-30까지만 자동 적용한다.
  • 혜택 규칙은 유효기간·출처·마지막 확인일을 가진 데이터로 관리한다.
  • 사용자 입력과 최근 결과는 서버 전송 없이 기기 안에만 저장한다.
  • main 병합·push·PR 없이 구현 브랜치에서 Strong Mode 검토를 기다린다.

마찰과 해결

  • Git 승인 한도 중단: 변경을 보존한 뒤 사용자 승인 후 같은 브랜치에서 재개했다.
  • 빠른 입력 상태 경합: 브라우저 재현 후 함수형 React 상태 업데이트로 수정했다.
  • 초기화 확인 UI 잔류: 초기화 시 확인 상태도 함께 닫도록 수정했다.
  • 신한 공식 URL 변경: 현재 공식 카드·해외이용 안내 URL로 교체했다.
  • Tab 키 자동 주입 제한: 의미 구조를 점검하고 실제 Safari 키보드 검증을 후속 작업으로 남겼다.

검증과 후속 작업

  • PASS 31개 테스트, 프로덕션 빌드, diff whitespace 검사
  • PASS 320·375·390px 무가로 스크롤, 콘솔 오류 없음
  • PASS 서버 중단 후 앱·입력 복원, 오프라인 재계산·상세 펼침
  • Strong Mode에서 iOS Safari 설치·VoiceOver·실제 키보드 포커스를 확인한다.
  • 승인 전에는 main 병합, push, PR, Pages 배포를 수행하지 않는다.
